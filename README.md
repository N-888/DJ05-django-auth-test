# Django Auth Test

Учебный проект на Django с кастомной моделью пользователя, регистрацией, входом, CSRF-защитой, группами, разрешениями и адаптивным интерфейсом на Bootstrap без CDN.

## Что демонстрирует проект
Проект показывает правильный учебный подход к кастомизации авторизации в Django: собственная модель пользователя, собственная форма регистрации, корректная настройка AUTH_USER_MODEL, безопасный logout через POST и CSRF, а также разграничение доступа через группы и разрешения.

## Основные возможности

- кастомная модель пользователя `CustomUser`
- уникальный `email`
- дополнительное поле `phone_number`
- удобные формы регистрации и входа
- автоматический вход после успешной регистрации
- защита всех форм через CSRF-токен
- разграничение доступа через группы и permissions
- отдельная защищенная страница списка пользователей
- адаптивный современный интерфейс
- локальный Bootstrap в `static/` без использования CDN
- подсветка активного пункта меню
- готовая интеграция с Django Admin

## Почему этот проект удачный

Проект получился не просто рабочим, а удобным, современным и аккуратным. Интерфейс адаптивен, формы читабельны, навигация понятна, а архитектура остается профессиональной и чистой. Код разделен по файлам и папкам логично, поэтому проект легко поддерживать, расширять и показывать как пример качественной учебной работы.

## Выгодные стороны решения

- используется `AbstractUser`, что является удобным способом расширения стандартного пользователя
- email сделан уникальным на уровне модели и формы
- телефон вынесен в отдельное поле и корректно валидируется
- статические файлы Bootstrap подключены локально, что удобно для учебных и офлайн-сценариев
- система прав уже подготовлена для дальнейшего расширения
- logout реализован безопасно и совместим со стандартным `LogoutView` Django
- дизайн выглядит современно и не требует фронтенд-фреймворков
